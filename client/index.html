The new Mountain Guarder Item Creator Tool (MGICTool) is out! Easily create a range of items from weapons to armor! Make custom offhand items using the "offhand" slot type!
In other news, the inventory is now mostly complete, and player and monster stats now apply. Next update will be on the actual effects of items (so don't go spamming the issues about it!).
  • Added the Mountain Guarder Item Creator Tool!
  • Modified item stats and added more items
  • Player and monster stats now apply
  • Items can only be equipped in their respective slot types now (Issue #83)

  • Added the rest of item images (Issue #81)
  • Patched crash exploit through socket.io (Issue #84)
